# Contributing

Thanks for your interest in contributing to VoiceAgent.

## Workflow

1. **Fork** the repo on GitHub
2. **Branch** in your fork (`feature/your-topic`)
3. **Commit** your change with a clear message (English preferred)
4. **Pull request** against `main` — describe what and why

Direct pushes to `main` are not possible — you need a PR.
